## Voraussetzungen:
- Java JDK 21 (oder höher)
- Gradle Wrapper (im Repository enthalten)

## Schritt-für-Schritt:
1. Repository klonen
2. Terminal im Projektverzeichnis öffnen
3. Starten mit:
    - Windows: `gradlew.bat run`
    - Linux/macOS: `./gradlew run`
4. Das Hauptmenü erscheint in der Konsole
